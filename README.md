# TNEU News

## Install

Ensure you have [Node.js](https://nodejs.org) version 4 or higher installed. Then run the following:

```
$ npm install tneu-news --save
```

## Usage

```javascript
const news = require('tneu-news');


```


## Development

### Tests

Used: chai, mocha

```
$ npm test
```

### Coverage

Used: istanbul

```
$ npm run coverage
```

### Linting

Used: eslint, eslint-config-google

```
$ npm run lint
```
